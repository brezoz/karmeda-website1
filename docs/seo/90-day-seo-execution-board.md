# 90-Day SEO Execution Board — SeragamRapih / PT Karmeda

> Board ini adalah jembatan antara PRD, master keyword map, dan cron SEO yang sudah aktif.
> Tujuannya: memastikan 90 hari ke depan ada urutan eksekusi yang jelas, terukur, dan tetap adaptif.

**Target utama:** dorong keyword komersial prioritas seragam kerja ke halaman 1 Google secara bertahap.  
**Source of truth keyword:** `docs/seo/master-keyword-map-seragamrapih.md`  
**Primary domain:** `seragamrapih.com`  
**Execution workspace:** `/home/marchel-H/tmp-karmeda-website1`

## Struktur 90 Hari

### Wave 1 — Money Page Consolidation (Hari 1–14)
Fokus:
- rapikan cluster keyword utama P1
- pastikan homepage + 3 money pages utama benar-benar sinkron dengan target intent
- audit gap internal link, FAQ, trust, CTA, schema, dan body depth

Target cluster:
- Cluster A — vendor seragam perusahaan / korporat
- Cluster B — konveksi seragam Tangerang
- Cluster C — wearpack safety / proyek
- Cluster D — homepage umbrella commercial intent

Done criteria:
- semua priority pages punya keyword focus yang lebih tegas
- internal linking dasar antar money pages lebih kuat
- ada daftar gap yang masih tersisa untuk support content

### Wave 2 — Support Content & Internal Link Expansion (Hari 15–42)
Fokus:
- bikin supporting articles yang langsung bantu money pages
- bangun link graph yang sengaja diarahkan ke cluster A/B/C/D
- hindari artikel generik yang tidak dorong intent beli

Target cluster:
- Cluster F — procurement / buying process
- article pendukung untuk cluster A/B/C

Done criteria:
- ada batch awal artikel support prioritas
- setiap artikel punya CTA dan internal link ke money page
- ada mapping article → target money page

### Wave 3 — Local Authority & Vertical Expansion (Hari 43–70)
Fokus:
- perkuat sinyal lokal Tangerang
- uji expansion ke Jabodetabek/vertical hanya jika evidence mendukung
- tambah proof, local relevance, dan page candidates yang justified

Target cluster:
- Cluster B — konveksi seragam Tangerang
- Cluster E — vertikal industri

Done criteria:
- local trust lebih kuat
- ada keputusan berbasis evidence apakah expand ke geo/vertical tertentu
- tidak ada expansion liar tanpa dasar

### Wave 4 — Refresh, Re-Rank, and Push to Page One (Hari 71–90)
Fokus:
- review keyword yang paling dekat Top 10
- refresh title/H1/body/internal links pada halaman yang paling menjanjikan
- gunakan evidence mingguan/bulanan dari cron untuk pivot bila perlu

Target cluster:
- semua cluster aktif dengan prioritas pada halaman paling dekat page one

Done criteria:
- ada shortlist keyword + halaman yang paling dekat target
- refresh dilakukan pada kandidat terbaik
- scorecard akhir 90 hari siap dipakai untuk next quarter

---

## Board Design Rules

1. Semua task harus mengacu ke keyword map hidup.
2. Semua task harus jelas target page atau target cluster-nya.
3. Task support content tidak boleh berdiri sendiri tanpa target money page.
4. Hanya task eksekusi yang benar-benar verified yang boleh ditutup done.
5. Cron boleh mengubah prioritas, tapi board tetap jadi kerangka kerja utamanya.
6. Jika evidence berubah, task berikutnya boleh diprioritaskan ulang tanpa mengubah tujuan utama.

## Measurement Rules

Weekly measure:
- keyword cluster paling layak didorong
- halaman prioritas minggu depan
- perubahan yang sudah dieksekusi
- risiko / hambatan
- apakah keyword map perlu update

Monthly measure:
- 3 cluster prioritas berikutnya
- asumsi yang harus ditantang
- fokus 30 hari berikutnya

End-of-90-day measure:
- keyword mana yang paling dekat / sudah masuk page one
- halaman mana yang paling menghasilkan momentum
- cluster mana yang harus digandakan / dikurangi
