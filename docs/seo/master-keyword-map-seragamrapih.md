## 1) Cara Pakai Dokumen Ini

### Prinsip utama
- Fokus ke keyword yang **dipakai calon pembeli**, bukan keyword ramai tapi lemah niat beli.
- Fokus ke **keyword komersial seragam kerja dan sejenisnya**.
- Kalau phrasing pasar berubah, map ini **harus ikut berubah**.
- Satu keyword utama harus punya **target page yang jelas**.
- Artikel support harus mendorong money page, bukan berdiri sendiri tanpa arah.

### Aturan prioritas
Gunakan label berikut:
- **P1** = paling dekat ke lead / uang / ranking impact
- **P2** = penting, tapi support atau second wave
- **P3** = long-tail / eksperimen / pendukung tambahan

### Aturan status
- **active** = sedang dikejar aktif
- **monitor** = dipantau, belum jadi fokus utama
- **build** = perlu halaman/cluster baru
- **refresh** = halaman ada, perlu diperkuat
- **hold** = ditunda sementara
